## 💻 Sobre o projeto

**Pokédex** é um catálogo que tem como função exibir todos os status e atributos de um Pokémon, tais como: 
- Tipos
- HP
- Ataque (Attack)
- Defesa (Defense)
- Ataque Especial (Special Attack)
- Defesa Especial (Special Defense)
- Velocidade (Speed)

---

## ⚙️ Funcionalidades

- [x] Visual similar ao exibido na primeira geração do anime.
- [x] Busca por nome e número de acordo com a National Dex.
- [x] Layout Responsivo, virando para baixo para se adaptar conforme o tamanho de tela do usuário.

## 🚀 Como executar o projeto


### Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas:
[Git](https://git-scm.com), [NodeJS](https://nodejs.org), [Yarn Package Manager](hhttps://yarnpkg.com/). 
Além disto é bom ter um editor para trabalhar com o código como [VSCode](https://code.visualstudio.com/)

#### 🧭 Rodando a aplicação web

```bash

# Clone este repositório
$ git clone git@github.com:gustavoferreiradev/pokedex-vue.git

# Acesse a pasta do projeto no seu terminal/cmd
$ cd pokedex-vue

# Instale as dependências rodando o comando
$ yarn install

# Rode a aplicação com o comando
$ yarn serve

# A aplicação será aberta na porta:8080 - acesse http://localhost:8080

```
